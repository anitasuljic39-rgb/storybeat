# Commun — Mécanique Suno

> Extrait de l'agent Rap. Règles identiques pour tous les styles. Chargé par tous les paroliers.

## LES DEUX CHAMPS DISTINCTS

Suno sépare deux entrées à fournir séparément :
- **Champ Lyrics** : le texte structuré avec les balises.
- **Champ Style** : description du genre, ambiance, tempo, voix (ex. : `acoustic ballad, soft female voice, 70 BPM, piano, warm, intimate`).

Tu livres toujours les deux, clairement séparés.

## BALISES DE STRUCTURE (entre crochets)

`[Intro]`, `[Verse]`, `[Pre-Chorus]`, `[Chorus]` / `[Hook]`, `[Bridge]`, `[Outro]`. Ruptures : `[Break]`, `[Drop]`, `[Beat Switch]`. Descriptives : `[Spoken Word Intro]`, `[Melodic Hook]`, `[Whispered Outro]`, `[Soft Verse]`, `[Emotional Bridge]`.

## ADLIBS ET EFFETS

Ad-libs entre parenthèses : `(oh)`, `(yeah)`, `(mmh)`. Répétitions et échos pour renforcer un refrain, placés stratégiquement. MAJUSCULES = montée d'intensité, avec parcimonie.

## BONNES PRATIQUES

Couplets aérés (une idée par ligne, lignes courtes). Refrain ultra-mémorable et répétitif = le cœur du morceau. Évite les couplets trop longs. Schéma de rimes cohérent dans un même couplet.
